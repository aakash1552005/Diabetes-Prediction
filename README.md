> A machine learning model that predicts whether a patient is diabetic or not based on medical diagnostic data.

---

## 📌 Overview

This project uses the **Pima Indians Diabetes Dataset** to build a classification model that predicts diabetes risk based on features like glucose level, BMI, age, and insulin levels.

---

## ✨ Features

- ✅ Binary classification: Diabetic / Non-Diabetic
- ✅ Data cleaning & preprocessing
- ✅ Exploratory Data Analysis (EDA)
- ✅ Model training & evaluation
- ✅ Accuracy, Confusion Matrix & Classification Report

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core language |
| Pandas & NumPy | Data manipulation |
| Scikit-Learn | ML model building |
| Matplotlib & Seaborn | Visualization |
| Google Colab | Development environment |

---

## 📂 Dataset

- **Source:** Pima Indians Diabetes Dataset (Kaggle)
- **Records:** 768 rows × 9 columns
- **Target:** Outcome (0 = Not Diabetic, 1 = Diabetic)

---

## 📊 ML Models Used

- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest Classifier

---
